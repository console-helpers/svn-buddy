
The "reparse" command
=====================


Reparses given revision.

Arguments
---------


* ``path`` - Working copy path [default: "``.``"]


Options
-------


* ``-r``, ``--revision=REVISION`` - Reparse specified revision


Examples
--------



.. code-block:: bash

   svn-buddy.phar reparse --revision 12345


Re-reads and reparses 12345 revision information.
